The com.clockwork.scene.mesh package contains utilities for reading from {@link com.clockwork.scene.mesh.IndexBuffer index buffers}. Several implementations are provided of the {@link com.clockwork.scene.mesh.IndexBuffer} class:
  • {@link com.clockwork.scene.mesh.IndexByteBuffer} - For reading 8-bit index buffers
  • {@link com.clockwork.scene.mesh.IndexShortBuffer} - For reading 16-bit index buffers
  • {@link com.clockwork.scene.mesh.IndexIntBuffer} - For reading 32-bit index buffers
  • {@link com.clockwork.scene.mesh.VirtualIndexBuffer} - For reading "virtual indices", for those meshes that do not have an index buffer
  • {@link com.clockwork.scene.mesh.WrappedIndexBuffer} - For converting from non-list based mode indices to list based
